# Archive

> *DROP'd entries. Preserved with reasons, not deleted.*

Entries in `archive/` were once active and have been demoted. Per the discipline in [`../STATUS_LEGEND.md`](../STATUS_LEGEND.md):

- A DROP'd entry is *demoted, not deleted*.
- Each entry records the **reason** for demotion.
- If a successor entry exists, the archive entry **links** to it.

The archive exists because re-inventing and re-rejecting the same idea is a failure mode. Recording why an idea was demoted is the antidote.

---

## What gets archived

Reasons (per [`../CONTRIBUTING.md`](../CONTRIBUTING.md)):

1. **The falsifier was run and failed**, and no fix path is in flight.
2. **A literature collision makes the claim** and reduces the entry to a restatement.
3. **A successor entry supersedes it.**
4. **The author can no longer state the claim precisely.**

Quiet deletion is forbidden. If an idea was once a claim, it stays in the historical record — the alternative is silent overclaiming about which ideas have been considered.

## Why this is a discipline file

A repo without an archive looks cleaner. It also lies: it pretends the path to the current claims was straight, that no ideas were tried and rejected, that the work emerged fully-formed.

The archive is the antidote. **A research program that has not rejected anything is not doing research.** Recording the rejections is part of the work.
